## 🚀 Як починати?

### 1. Відкрий папку розділу
Починай з `chapter-01-zminni/zavdannya.js`

### 2. Читай завдання
Кожен файл `zavdannya.js` містить:
- **Сюжет** — що відбувається у галактиці
- **Пояснення** — що потрібно знати
- **Завдання** — що потрібно написати
- Позначки `???` — саме тут ти пишеш свій код!

### 3. Виконуй завдання
Замість `???` пиши свій JavaScript-код.

### 4. Перевіряй результат
```bash
node chapter-01-zminni/testy.js
```

### 5. Переходь далі
Коли всі тести зелені ✅ — переходь до наступного розділу!
